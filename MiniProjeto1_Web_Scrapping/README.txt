Web Scraper de Notícias do G1

Descrição

Este é um script em Python que usa as bibliotecas `requests` e `BeautifulSoup` para extrair as manchetes principais e seus links da página inicial do portal de notícias G1.

Como Executar

Para rodar este projeto, siga os passos abaixo.

1. Baixe os Arquivos
Faça o download dos arquivos deste projeto (`web_scrapping.py` e `requirements.txt`) e salve-os em uma mesma pasta no seu computador.

2. Instale as Dependências
Abra o seu terminal, navegue até a pasta onde você salvou os arquivos e execute o seguinte comando para instalar as bibliotecas necessárias:
```bash pip install -r requirements.txt```

### 3. Execute o Script
No mesmo terminal, execute o script com o comando:
```bash python web_scrapping.py```

O terminal irá então exibir a lista de manchetes e links extraídos do site.
